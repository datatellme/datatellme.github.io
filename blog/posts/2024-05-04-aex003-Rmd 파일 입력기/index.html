    챗 GPT 답변

    • A01: 물론이죠! Rmd(R Markdown) 문서를 작성하는 입력기를 만들어 보겠습니다. 아래는 파이썬으로 작성된 Tkinter를 사용한 간단한 Rmd 문서 입력기입니다. 사용자가 텍스트를 작성하고 저장할 수 있으며, 저장 시간이 Rmd 파일의 주석으로 추가됩니다.
    import tkinter as tk
from tkinter import messagebox
import os
from datetime import datetime

def save_text_to_file():
    text_content = text_entry.get("1.0", tk.END)  # 텍스트 입력란의 내용 가져오기

    # 출력 디렉토리 및 파일 경로 설정
    output_directory = r'C:\gppj\gppj01\i'
    output_file = 'document.Rmd'
    output_path = os.path.join(output_directory, output_file)

    # 현재 시간
    current_time = datetime.now().strftime("%Y-%m-%d %H:%M:%S")

    # Rmd 문서 내용 생성
    rmd_content = f'''---
title: "Rmd 문서"
author: "사용자"
date: "{current_time}"
output: html_document
---

# 내용

{text_content}
'''

    # Rmd 파일에 저장
    with open(output_path, 'w', encoding='utf-8') as file:
        file.write(rmd_content)

    # 저장 완료 메시지 출력
    messagebox.showinfo("저장 완료", "Rmd 문서가 저장되었습니다.")

# Tkinter GUI 생성
root = tk.Tk()
root.title("Rmd 문서 작성기")
root.geometry("400x300")

# 텍스트 입력 위젯 생성
text_entry = tk.Text(root)
text_entry.pack(fill="both", expand=True)

# 저장 버튼 생성
save_button = tk.Button(root, text="저장", command=save_text_to_file)
save_button.pack()

root.mainloop()
    • 이 코드를 실행하면 Tkinter 기반의 GUI 창이 열리고, 사용자가 Rmd 문서 내용을 입력할 수 있습니다. "저장" 버튼을 클릭하면 입력한 내용이 지정된 디렉토리에 "document.Rmd" 파일로 저장됩니다. 파일에는 입력된 내용과 현재 시간이 포함됩니다.
