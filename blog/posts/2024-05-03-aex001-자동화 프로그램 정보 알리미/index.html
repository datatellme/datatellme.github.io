    자동화 프로그램 정보 알리미(gppj0104)

    들어가며(Introduction)

    목표(Goal)

    • 파이썬으로 자동화 프로그램을 진행 할때 프로그램 진행 시간을 초단위로 확인할 수 있고 여러개의 독립적인 프로그램들을 순차적으로 진행되는지를 확인 할 수 있는 정보 알리미 프로그램을 만들는 것이 목표입니다.

    목적(Why)

    • 이것을 만드는 목적은 각 장에서 만든 작은 단위의 프로그램들을 연결할 때 어떤 순서로 연결되었는지와 연결된 여러프로그램들의 구동시간이 총 얼마기 소요되었는지의 정보를 확인하기 위해서 만드는 것입니다.

    • 파이썬 프로그램은 컴퓨터를 통하여 빠른 시간에 분석하기 때문에 어디로 가나 서울만 가면 된다는 속담과 같이 결과물을 산출하기 위해서 여러단계의 과정을 거치더라도 많은 시간이 소요되지는 않습니다.

    • 이렇게 여러 단계의 일을 하나의 프로젝트로 묶기 위해서는 어떠한 단계로 분석했는지 어떤 코드를 사용했는지를 확인할 필요가 있습니다. 이에 4장에서 프로젝트 01을 만들어 면서 구체적으로 적용해 보도록 하겠습니다.

    방법(How)

    • 질문01: 텍스트 파일 내용을 기록하고 프로그램 구동시간 기록하기

    • 질문02: 텍스트 파일 작성한 날짜와 시간 기록하기

    • Chat GPT에게 질문(프롬프트)하는 형식은 제 2장에서 구체적으로 다루기로 하고 일단 아래와 같이 Q01의 질문을 만들에 프롬프트에 입력하면 바로 정답을 줍니다. 하지만 원하는 정답이 아닌 경우 Q02의 질문처럼 다시 결과 코드를 수정하도록 해야 합니다.

    • 같은 프롬프트를 입력하더라도 Chat GPT의 코딩은 입력할 때마다 조금씩 다르게 나오나 프로그램을 돌렸을 때 최종 결과물은 같게 산출됩니다.

    • 데이터 분석에서 사용되는 금언인 '금을 넣으면 금이나온다.'는 말과 같이 프롬프트를 잘 작성하여야 좋은 코딩 결과를 기대할 수 있습니다. 즉 파이썬 언어에 작성하지는 못하지만 해석 할 정도의 수준은 되어야 합니다.

    1단계 질문01(프롬프트)

    Q01: 너는 10년차 파이썬 프로그래머처럼 PEP 8스타일 가이드를 따라 파이썬 언어를 사용해서 파이썬 대화창에 

출력되는 내용을 텍스트파일로 저장하는 기능을 수행하는 코드를 생성해줘. 

예를 들어 프린트한 내용의 텍스트 파일은 directory_out = r'C:\Temp\i'에 inf_output.txt로 만들어 주는거야.

파이썬 대화창에 프린트할 내용은 다음과 같아.

"주제: 

Why(목적): 

How(테스크 나누기):
1) step01
2) step02
3) step03
4) step04


What(산출): 

작성일 : 2023.00.00.
작성자: 이름"

프린트할 때 sys.stdout을 사용해서 작성해줘.

시작 시간과 끝 시간을 계산해서 프린트 시간도 초단위로 inf_output.txt에 적어줘.

def 함수를 사용하지 않고 작성해줘.

os를 사용해서 작성해줘.

    2단계 질문02(프롬프트)

    Q02: 아래 코드 참고해서 텍스트파일을 작성하고 저장할때의 현제 시간이 기록되게 수정 부탁해.

아래

# 현재 날짜와 시간을 가져옴
current_datetime = datetime.now()

# 날짜와 시간을 각각 변수에 저장
year = current_datetime.year
month = current_datetime.month
day = current_datetime.day
hour = current_datetime.hour
minute = current_datetime.minute
second = current_datetime.second

# 현재 작업한 시간 텍스트 파일에 작성
with open(output_path, 'a', encoding='utf-8') as file:
    file.write(f"작성 년도: {year}.")
    file.write(f"{month}.")
    file.write(f"{day}.\n")
    file.write(f"현재 시간: {hour}:{minute}:{second}\n\n")

    결과(What)

    • 위에 지시한 프롬프트 질문을 카피해서 Chat GPT에게 물어보면 물어 볼때마다 조금씩 다른 내용의 답변을 하는 것을 볼 수 있습니다.
    • 그래서 우리는 최소한의 파이썬 코딩을 해석할 정도의 지식이 필요합니다.
    • 파이썬 문법을 알고 있어야 결과를 해석하고 오류를 수정할 수 있는 것입니다.

    1단계 Chat GPT 답변01

    import os
import sys
import time

# 출력할 내용
output_text = """주제: 

Why(목적): 

How(테스크 나누기):
1) step01
2) step02
3) step03
4) step04


What(산출): 

작성일 : 2023.00.00.
작성자: 이름"""

# 시작 시간 기록
start_time = time.time()

# 출력 내용을 텍스트 파일에 저장
directory_out = r'C:\Temp\i'
file_path = os.path.join(directory_out, 'inf_output.txt')

with open(file_path, 'w') as file:
    # 표준 출력 임시 저장
    sys_stdout = sys.stdout
    sys.stdout = file

    # 출력
    print(output_text)

    # 표준 출력을 원래대로 복원
    sys.stdout = sys_stdout

# 끝 시간 기록
end_time = time.time()

# 프린트 시간을 초 단위로 계산하여 텍스트 파일에 추가
with open(file_path, 'a') as file:
    print(f"\n프린트 시간: {end_time - start_time:.2f} 초", file=file)

    결과 디버깅 및 수정

    • 사용하는 패키를들을 맨 위에 위치시켰습니다.
    • 텍스트 프린트할 내용을 구분하기위해 '----' 실선을 추가했습니다.
    • 자동화 파이썬 코드 작성 공간을 추가하고 step별로 구분하여 시각적으로 확인 가능하게 하였습니다.
    • 자동화 파이썬 코드 작성 공간에는 단계적으로 프로그램을 구동되게 위치시키는 것입니다.
    • 그리고 구동된 프로그램이의 구동하는데 걸린 시간과 구동하는 현재 시간을 알려주고 프로그램 입니다.
    • 이러한 업무 자동화 정보 알리미는 자동화 프로그램을 만들때 어떤 모듈을 사용해서 어떤 절차에 따라 자동화 되는지를 텍스트파일로 알려주기위한 것입니다.
