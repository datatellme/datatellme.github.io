    Q01: 너는 10년차 파이썬 프로그래머처럼 PEP 8스타일 가이드를 따라 파이썬 언어를 사용해서 MySql 명령어 

입력기의 출력 결과를 csv파일로 저장하는 기능을 수행하는 코드를 추가해서 수정 부탁해. 

예를 들어 쿼리 출력 결과를 directory = r'C:\gppj\gppj07_my\y' 안에 test01.csv 파일로 저장하는 거야.

Tkinter를 사용하여 GUI 창에서 실행 버튼을 누르면 test01.csv 이 저장 되며 '저장이 완료되었습니다' 라는 

메세지를 출력 되도록 수정 부탁해.

import tkinter as tk
import pymysql

def execute_sql():
    # MySQL 데이터베이스 연결 설정
    connection = pymysql.connect(
        host='localhost',  # 호스트 이름
        user='root',   # 사용자 이름
        password='1234',   # 비밀번호
        database='test01_db'  # 데이터베이스 이름
    )

    try:
        with connection.cursor() as cursor:
            # 입력된 SQL 쿼리 가져오기
            sql_query = sql_entry.get("1.0", tk.END).strip()

            # SQL 쿼리 실행
            cursor.execute(sql_query)

            # 결과 가져오기
            result = cursor.fetchall()
            for row in result:
                print(row)
    except Exception as e:
        print("SQL 실행 오류:", e)
    finally:
        # 연결 닫기
        connection.close()

# Tkinter GUI 생성
root = tk.Tk()
root.title("MySQL 명령어 입력기: 예-SELECT * FROM test01_db.collection01;")
root.geometry("400x300")

# SQL 명령어 입력 창 생성
sql_entry = tk.Text(root, wrap=tk.WORD, height=10)
sql_entry.pack(fill=tk.BOTH, expand=True)

# 실행 버튼 생성
execute_button = tk.Button(root, text="실행", command=execute_sql)
execute_button.pack()

root.mainloop()

    다음은 코드에 CSV 파일로 결과를 저장하는 기능을 추가한 수정된 버전입니다. 결과는 CSV 파일로 저장될 

것이며, 저장이 완료되면 메시지를 출력합니다.

import tkinter as tk
import pymysql
import csv

def execute_sql():
    # MySQL 데이터베이스 연결 설정
    connection = pymysql.connect(
        host='localhost',  # 호스트 이름
        user='root',   # 사용자 이름
        password='1234',   # 비밀번호
        database='test01_db'  # 데이터베이스 이름
    )

    try:
        with connection.cursor() as cursor:
            # 입력된 SQL 쿼리 가져오기
            sql_query = sql_entry.get("1.0", tk.END).strip()

            # SQL 쿼리 실행
            cursor.execute(sql_query)

            # 결과 가져오기
            result = cursor.fetchall()
            for row in result:
                print(row)

            # 결과를 CSV 파일로 저장
            save_to_csv(result)

            # 저장 완료 메시지 출력
            tk.messagebox.showinfo("저장 완료", "결과를 CSV 파일로 저장했습니다.")
    except Exception as e:
        print("SQL 실행 오류:", e)
    finally:
        # 연결 닫기
        connection.close()

def save_to_csv(result):
    # CSV 파일 경로
    csv_file = r'C:\gppj\gppj07_my\y\test01.csv'

    # 결과를 CSV 파일에 쓰기
    with open(csv_file, 'w', newline='') as file:
        writer = csv.writer(file)
        writer.writerows(result)

# Tkinter GUI 생성
root = tk.Tk()
root.title("MySQL 명령어 입력기: 예-SELECT * FROM test01_db.collection01;")
root.geometry("400x300")

# SQL 명령어 입력 창 생성
sql_entry = tk.Text(root, wrap=tk.WORD, height=10)
sql_entry.pack(fill=tk.BOTH, expand=True)

# 실행 버튼 생성
execute_button = tk.Button(root, text="실행", command=execute_sql)
execute_button.pack()

root.mainloop()
이제 쿼리 실행 결과가 CSV 파일로 저장되며, 저장이 완료되면 메시지가 표시됩니다.

    결과 디버깅 및 수정

    • Chat GPT 위의 답변에서 "from tkinter import filedialog, messagebox" 누락되어 에라 메세지가

    발생하였습니다.

    • 이러한 에라는 코딩하는 우리가 체크하여 수정해 넣는 것이 좋습니다.

    • 에라 메세지를 카피해서 질문하면 올바른 답을 줄 수도 있지만 계속 잘못된 답을 주는 경우도 있습니다.
