    Chat GPT 답변

    A01: Scrapy를 사용하여 한국은행 기준금리 및 여수신금리를 스크레이핑하는 Spider 코드를 작성하겠습니다. Scrapy 프로젝트를 먼저 생성한 후, items.py 파일을 정의하고, 스파이더를 작성하는 방법을 안내하겠습니다.

    1. Scrapy 프로젝트 생성 터미널에서 다음 명령어를 실행하여 Scrapy 프로젝트를 생성합니다.
    scrapy startproject kor_bank_data
    1. items.py 파일 정의

    kor_bank_data 디렉토리 안에 있는 items.py 파일을 열고 다음과 같이 수정합니다.

    import scrapy

class KorBankDataItem(scrapy.Item):
    date = scrapy.Field()
    base_rate = scrapy.Field()
    deposit_rate = scrapy.Field()
    loan_rate = scrapy.Field()
    1. Spider 작성

    kor_bank_data/spiders 디렉토리에 collect_base_rate.py 파일을 생성하고, 다음 코드를 작성합니다.

    import scrapy
from kor_bank_data.items import KorBankDataItem

class CollectBaseRateSpider(scrapy.Spider):
    name = 'collect_base_rate'
    allowed_domains = ['ecos.bok.or.kr']

    # 한국은행 Open API 서비스 URL
    start_urls = [
        'https://ecos.bok.or.kr/api/StatisticSearch/YOUR_API_KEY/json/kr/1/1000/060Y001/DD/20230101/20231231/0101000'
    ]

    def parse(self, response):
        # JSON 응답을 파싱
        data = response.json()
        rows = data.get('StatisticSearch', {}).get('row', [])

        for row in rows:
            item = KorBankDataItem()
            item['date'] = row.get('TIME')
            item['base_rate'] = row.get('DATA_VALUE')

            # 예금 금리 및 대출 금리 데이터를 가져오는 로직 추가 필요
            # 예제에서는 기본적으로 기준금리만 가져오고, 실제 API 문서 참고 필요
            # item['deposit_rate'] = row.get('DEPOSIT_RATE')
            # item['loan_rate'] = row.get('LOAN_RATE')

            yield item
    1. settings.py 파일 수정

    kor_bank_data/settings.py 파일에서 파이프라인 설정을 활성화합니다.

    # Enable item pipelines
ITEM_PIPELINES = {
   'kor_bank_data.pipelines.KorBankDataPipeline': 300,

    1. pipelines.py 파일 작성

    kor_bank_data/pipelines.py 파일을 열고 다음 코드를 작성합니다.

    import pymysql

class KorBankDataPipeline:
    def open_spider(self, spider):
        self.connection = pymysql.connect(
            host='localhost',
            user='root',
            password='password',
            database='kor_bank_data_db',
            charset='utf8mb4'
        )
        self.cursor = self.connection.cursor()
        self.create_table()

    def close_spider(self, spider):
        self.connection.close()

    def create_table(self):
        create_table_query = """
        CREATE TABLE IF NOT EXISTS base_rate_data (
            date DATE,
            base_rate FLOAT,
            deposit_rate FLOAT,
            loan_rate FLOAT,
            PRIMARY KEY (date)
        )
        """
        self.cursor.execute(create_table_query)
        self.connection.commit()

    def process_item(self, item, spider):
        insert_query = """
        INSERT INTO base_rate_data (date, base_rate, deposit_rate, loan_rate)
        VALUES (%s, %s, %s, %s)
        ON DUPLICATE KEY UPDATE
        base_rate=VALUES(base_rate), deposit_rate=VALUES(deposit_rate), loan_rate=VALUES(loan_rate)
        """
        self.cursor.execute(insert_query, (
            item['date'],
            item['base_rate'],
            item.get('deposit_rate', None),
            item.get('loan_rate', None)
        ))
        self.connection.commit()
        return item
    1. 데이터베이스 설정

    MySQL 데이터베이스를 설정하고 kor_bank_data_db 데이터베이스를 생성합니다.

    CREATE DATABASE kor_bank_data_db;
    1. 스크레이핑 실행

    터미널에서 다음 명령어를 실행하여 스크레이핑을 시작합니다.

    scrapy crawl collect_base_rate
    이렇게 하면 스크래피를 사용하여 한국은행 기준금리 및 여수신금리를 수집하고 MySQL 데이터베이스에 저장할 수 있습니다. API 키를 포함한 URL을 수정하여 실제 데이터 수집이 가능하도록 하세요.
