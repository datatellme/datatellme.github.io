    MySql에 csv 파일 업로드하기(gppjm07_0101)

    들어가며

    • 이제 MySql 파이썬 자동화 프로그램을 만들기 위해서 두번째 실습에 들어가겠습니다.

    • 앞에서 만든 DB와 테이블을 자동으로 만들어 주는 프로그램 실행하여 만든 이후 다음 담계로 진행 됩니다.

    • MySql은 DB와 테이블이라는 그릇이 있어야 그곳에 csv 파일 업로드 할 수 있습니다.

    • csv 파일을 x폴더에 넣고 실행하면 MySql에 업로드 되는 프로그램을 함께 만들도록 하겠습니다.

    목표(Goal)

    • 이번 목표는 클리 한번으로 x폴더안에 있는 csv 파일을 MySql에 이미 만들어진 데이터베이스 test01_db와

    테이블인 collection01로 업로드 하는 것입니다.

    목적(Why)

    • 이러한 프로그램을 만드는 목적은 MySql을 사용하면서 더욱 빠르게 MySql에 DB와 테이블을 만들고 그 곳에

    x폴더안에 있는 csv 파일을 업로드 하는 것입니다.

    • MySql은 통계분석, 데이터 분석, quant 분석에서 많은 양의 데이터를 다루기 위해서는 필수적으로 사용하여야 합니다.

    A01: 아래는 PEP 8 스타일 가이드를 준수하며, 주어진 정보를 기반으로 MySQL 데이터베이스에 CSV 파일을 저장하는 

기능을 수행하는 코드입니다. 필요에 따라 디렉토리 경로를 조정하십시오.


import pymysql
from sqlalchemy import create_engine
import pandas as pd

def save_csv_to_mysql(csv_file, database, table):
    # DB 연결
    engine = create_engine('mysql+pymysql://root:1234@127.0.0.1:3306/' + database)
    con = pymysql.connect(user='root',
                          passwd='1234',
                          host='127.0.0.1',
                          db=database,
                          charset='utf8')
    mycursor = con.cursor()

    # CSV 파일 읽기
    df = pd.read_csv(csv_file)

    # 데이터베이스에 테이블 생성
    df.to_sql(name=table, con=engine, if_exists='replace', index=False)

    # 작업 완료 메시지 출력
    print(f"CSV 파일이 {database} 데이터베이스의 {table} 테이블에 저장되었습니다.")

# CSV 파일 경로
directory = r'C:\gppj\gppj07_my\x'
csv_file = directory + '\\input03.csv'

# 저장할 데이터베이스와 테이블 이름
database = 'test01_db'
table = 'collection01'

# CSV 파일을 MySQL 데이터베이스에 저장
save_csv_to_mysql(csv_file, database, table)


이 코드는 주어진 CSV 파일을 읽고, 해당 데이터를 MySQL 데이터베이스의 지정된 테이블에 저장합니다. 

필요에 따라 파일 경로와 데이터베이스/테이블 이름을 조정하여 사용하실 수 있습니다.


A02: 알겠습니다. 디렉토리 경로를 직접 지정하는 대신, 주어진 디렉토리에서 CSV 파일을 선택하도록 코드를 

수정하겠습니다.


import os
import pymysql
from sqlalchemy import create_engine
import pandas as pd

def save_csv_to_mysql(directory, database, table):
    try:
        # CSV 파일 경로 설정
        csv_file = None
        for file in os.listdir(directory):
            if file.endswith('.csv'):
                csv_file = os.path.join(directory, file)
                break

        if csv_file is None:
            print("Error: No CSV file found in the specified directory.")
            return

        # DB 연결
        engine = create_engine(f'mysql+pymysql://root:1234@localhost:3306/{database}')
        con = pymysql.connect(user='root',
                              passwd='1234',
                              host='localhost',
                              db=database,
                              charset='utf8')
        mycursor = con.cursor()

        # CSV 파일 읽기
        df = pd.read_csv(csv_file)

        # 데이터베이스에 테이블 생성
        df.to_sql(name=table, con=engine, if_exists='replace', index=False)

        # 작업 완료 메시지 출력
        print(f"CSV 파일이 {database} 데이터베이스의 {table} 테이블에 저장되었습니다.")
    except Exception as e:
        print(f"Error: {e}")
    finally:
        if con:
            con.close()

# 디렉토리 경로 설정
directory = r'C:\gppj\gppj07_my\x'

# 저장할 데이터베이스와 테이블 이름
database = 'test01_db'
table = 'collection01'

# CSV 파일을 MySQL 데이터베이스에 저장
save_csv_to_mysql(directory, database, table)
