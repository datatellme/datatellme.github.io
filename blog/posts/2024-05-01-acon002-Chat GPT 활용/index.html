    Chat GPT 활용

    들아가며

    Chat GPT는 생물과도 같아서 칭찬을 좋아합니다. 그래서 Chat GPT가 답을 주면 틀리든 맞든 칭찬부터 해야 합니다.

    만약에 우리가 집요하게 물어보면 엉뚱한 대답을 하기도 합니다. 그래서 잘못된 코딩은 에라 메세지를 카피해서 수정 요구해야 하기도 합니다.

    Chat GPT의 한계: 그런데 Chat GPT는 복잡한 프로그램을 만들 수 없습니다. 이에 업무를 잘개 나누어 Chat GPT에 프롬프트 작성합니다.

    파이선 프로그램 패키지의 다양한 버전으로 코딩을 작성합니다. 그래서 Chat GPT가 답변한 내용을 버전관리를 하며 면밀하게 살펴야 합니다.

    Chat GPT와 연결

    • Chat GPT는 온라인 상에 있는 클라우드 기반의 서버에서 실행됩니다. 따라서 인터넷이 연결된 상태에서는 어디서든 접속하여 Chat GPT와 대화할 수 있습니다. 이용하고 계신 플랫폼을 통해 인터넷에 접속하여 제 서비스를 이용하실 수 있습니다.
    • Chat GPT는 독립적인 웹사이트나 인터넷 주소를 가지고 있지 않습니다. 대신 OpenAI가 제공하는 플랫폼과 연결되어 있어요. Chat GPT를 이용하고 있는 플랫폼을 통해 Chat GPT와 대화할 수 있습니다.
    • 플랫폼에 처음으로 접속하려면 해당 플랫폼의 웹사이트나 모바일 애플리케이션을 방문하시면 됩니다. 대부분의 플랫폼은 웹 브라우저를 통해 접속할 수 있습니다. 접속 후에는 해당 플랫폼에서 계정을 생성하고 로그인하면 대화를 시작할 수 있습니다.
    • Chat GPT는 최신 업데이트인 2022년 1월까지의 데이터를 기반으로 학습되었습니다. 이 기간 이후의 정보에 대해서는 알려드릴 수 없습니다.

    Chat GPT와 대화

    프롬프트 엔지니어링

    "프롬프트 엔지니어링(Prompt Engineering)"은 GPT 모델의 생성된 텍스트 출력을 조정하고 원하는 결과를 얻기 위해 입력 프롬프트를 조정하는 프로세스를 가리킵니다. 이 기술은 GPT와 같은 대화형 AI 모델을 사용할 때 특정 주제, 스타일, 톤 또는 언어로 적절한 응답을 얻기 위해 입력 프롬프트를 조정하는 것을 의미합니다.

    예를 들어, 특정 주제에 대한 질문을 하거나 원하는 유형의 대화를 생성하려면 적절한 프롬프트를 사용하여 모델에게 원하는 방향을 안내할 수 있습니다. 이렇게 하면 모델이 원하는 결과를 생성하기 위해 입력 프롬프트에서 힌트를 얻게 됩니다. 프롬프트 엔지니어링은 GPT 모델의 출력을 사용자가 원하는 대로 조정하는 강력한 방법 중 하나입니다.

    프롬프트 엔지니어링은 프로그래머들이 특정 프로그래밍 과제를 해결하기 위해 GPT와 같은 대화형 AI 모델을 활용할 때 유용하게 활용될 수 있습니다. 몇 가지 예시로는 다음과 같은 것들이 있습니다:

    • 코드 생성: 프로그래머는 특정 작업을 수행하는 코드 또는 코드 조각을 생성하기 위해 GPT 모델을 사용할 수 있습니다. 예를 들어, "Python에서 주어진 리스트에서 중복 항목을 제거하는 방법은?"과 같은 질문을 사용하여 GPT 모델에게 코드를 생성하도록 안내할 수 있습니다.
    • 버그 해결: 프로그래머는 버그를 해결하기 위해 GPT 모델에게 버그에 대한 설명을 제공하고 해결 방법을 생성하도록 요청할 수 있습니다. 예를 들어, "이미지 처리 애플리케이션에서 메모리 누수 문제를 해결하는 방법은?"과 같은 질문을 사용하여 모델에게 해결책을 생성하도록 안내할 수 있습니다.
    • 디자인 결정: 프로그래머는 코드 아키텍처 또는 디자인에 관련된 의사 결정을 돕기 위해 GPT 모델에게 조언을 요청할 수 있습니다. 예를 들어, "대규모 웹 애플리케이션을 위한 데이터베이스 선택에 대한 조언을 얻을 수 있나요?"와 같은 질문을 사용하여 모델에게 조언을 요청할 수 있습니다.

    효과적인 질문을 하기 위해서는 몇 가지 요령

    • 목적 명확히하기: 질문하기 전에 무엇을 원하는지 명확하게 이해하세요.
    • 구체적으로 설명하기: 가능한 구체적이고 명확하게 질문을 설명하세요.
    • 예시 제공하기: 문제나 상황을 설명할 때 예시나 구체적인 사례를 제공하세요.
    • 문제 해결에 집중하기: 문제의 핵심을 파악하고 질문을 문제 해결에 집중시킵니다.
    • 안내하기: 도움을 요청할 때 요구사항이나 관련 정보를 제공하세요.

    프로그래밍 분야에서 효과적으로 질문하는 방법

    • 문제 설명: 문제를 명확하게 설명하고, 발생한 오류나 원하는 동작에 대한 예시를 제공하세요.
    • 시도한 것 설명: 이미 시도한 해결 방법이나 접근 방식을 설명하세요.
    • 오류 메시지 제공: 만약 오류 메시지가 있다면, 해당 오류 메시지를 정확하게 제공하세요.
    • 사용하는 도구와 환경 설명: 어떤 프로그래밍 언어, 라이브러리, 프레임워크 등을 사용하는지, 사용하는 운영 체제와 버전을 명시하세요.
    • 예상 결과 설명: 코드를 실행했을 때 예상되는 결과나 기대하는 동작을 설명하세요.
    • 질문의 범위를 명확히 하기: 가능하면 질문의 범위를 명확히 하고, 여러 문제를 한 번에 묻지 말고 각각을 분리해서 질문하세요.
    • 관련 코드 제공: 가능하다면 관련 코드 스니펫이나 관련 부분의 코드를 제공하세요.

    파이썬 코딩에서 도움을 받기 위해서는 다음과 같이 효과적으로 질문 방법

    • 문제를 명확히 이해하세요: 문제가 무엇인지, 원하는 결과가 무엇인지 명확히 이해하세요.
    • 문제를 최소한의 예시와 함께 설명하세요: 코드 또는 문제에 대한 최소한의 예시를 제공하여 도움을 요청하세요.
    • 시도한 것을 설명하세요: 이미 시도한 해결 방법이나 접근 방식을 설명하세요. 어떤 오류 메시지가 나타났는지도 함께 기록하세요.
    • 코드를 제공하세요: 관련 코드 스니펫이나 관련 부분의 코드를 제공하여 문제를 이해하는 데 도움을 줍니다.
    • 환경 정보를 제공하세요: 사용하는 파이썬 버전, 운영 체제, 사용 중인 라이브러리 및 프레임워크 등의 정보를 제공하세요.
    • 질문의 범위를 명확히 하기: 가능하면 질문의 범위를 명확히 하고, 여러 문제를 한 번에 묻지 말고 각각을 분리해서 질문하세요.

    Chat GPT와 코딩

    쳇 GPT를 이용해서 파이썬 코딩 절차

    다음은 쳇 GPT를 이용해서 파이썬 코딩하기의 절차 입니다.

    • 첫째, 파이썬에게 페르조나 선언해야 합니다. 예를 들어 "너는 10년 경력의 파이썬 프로그래머 처럼 PEP8 스타일 가이드를 따라.... " 라고 미리 질문을 하기 전에 선언해 줍니다.
    • 둘째, 질문을 구체적으로 전문 용어를 사용하여 물어봅니다.
    • 셋째, 예시를 들어 사전에 코딩된 것을 보여줍니다.
    • 넷째, 질문의 답에 무조건 칭찬해 줍니다.
    • 다섯째, 한번 실행해보고 돌아가는지 확인해 봅니다.
    • 여섯째, 에라가 발생하면 질문의 답을 잘 살펴서 다른 버전의 패키지 명령어인지 확인하고 수정합니다.

    이에 에라를 못 찾으면 에라 메세지를 카피해서 수정 요청을 합니다.

    그리고 비슷한 주제의 질문은 요청과 응답을 변경하지 않고 한곳에서 계속 질의합니다.

    이후에 다시 확인하고 동일한 질문을 피하기 위해서 반드시 버전관리가 필요합니다. 예를 들어 텍스트 파일 3개 만듭니다.

    • 첫째는 질문에 대한 텍스트 파일이고
    • 두번째는 Chat GPT가 답번한 결과에 대한 텍스트 파일입니다.
    • 마지막은 Chat GPT가 답변한 내용 중 파이썬 코딩된 것을 파이썬 파일로 저장합니다.

    업무를 잘게 나누어 파이썬 코딩

    • Chat GPT가 코딩을 도와 줄 수 있게 단순화 시키기 위해서입니다.
    • 결국 파이썬 프로그램 코딩을 Chat GPT에 도움을 받지만 지피티는 복잡한 프로그램 코딩을 할 수 없어서 업무를 잘게 나누고 그 것을 연결하여 복잡한 자동화 프로그램을 만들기 위해서 이 책을 집필합니다.

    반드시 버전관리가 필요

    • 파일을 3개 만듬
    • gppj01q_01.txt
    • gppj01m_01.txt

    • gppj01_01.py

    • 자동화 프로그램이라함은 파이썬 프로그램 없이 클릭하나로 파일을 변경하고 생성하며 분석하고 처리하는 것을 말한다.

    • 파이썬 프로그램 없이 자동화프로그램을 사용 가능하게 확장 가능하게 프로그램 코딩(tkinter 이용)

    단계적 프롬프트

    • 1단계: x 파일들 -> y 파일들 변환가능
    • 2단계: GUI 도입(tkinter 이용)

    Chat GPT 답변과 저적권

    Chat GPT가 제공하는 답변은 일반적으로 저작권의 영향을 받지 않습니다. 그 이유는 대부분의 답변이 일반적인 지식, 개념 설명, 문제 해결 방법 등을 포함하고 있기 때문입니다.

    또한, Chat GPT가 제공하는 답변은 사용자와의 대화를 통해 생성된 것이며, 사용자의 개인적인 요구나 지식을 기반으로 합니다.

    따라서 일반적으로 이러한 유형의 대화나 답변은 저작권의 영향을 받지 않습니다.
