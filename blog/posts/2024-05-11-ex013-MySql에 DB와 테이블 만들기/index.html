    MySql에 DB와 테이블 만들기 자동화 프로그램(gppjm07_00)

    들어가며

    이제 MySql 파이썬 자동화 프로그램을 만들기 위해서 첫번째 실습에 들어가겠습니다.

    MySql에서 DB와 테이블을 만드려면 자체 벤치마크를 사용하여 sql 쿼리 입력하여 만들어야 합니다.

    sql 문법을 잘모르지만 클릭하나로 DB와 테이블을 자동으로 만들어 주는 프로그램이 필요하여 만들게 되었습니다.

    MySql은 DB와 테이블이라는 그릇이 있어야 그곳에 csv 파일을 업로드 할 수 있습니다.

    목표(Goal)

    • 첫번째 목표는 클릭 한번으로 MySql DB와 test01_db와 MySql 테이블인 collection01을 만들고 임의의 컬럼을

    column1 column2를 만들고 DATATYPE을 각각 FLOAT과 VARCHAR(30)를 만드는 것입니다.

    목적(Why)

    • 이러한 프로그램을 만드는 목적은 MySql을 사용하면서 한번의 클릭으로 MySql에 DB와 테이블을 만들기 위한 것입니다.

    • MySql은 통계분석, 데이터 분석, quant 분석에서 많은 양의 데이터를 다루기 위해서는 필수적으로 사용해야 합니다.

    질문 작성 방법(How)

    • 질문 2개를 대화 하듯 하면서 답을 듣고 파이썬을 실행해 봅니다

    • 파이썬 실행시 에라 메세지가 뜨면 코드를 잘 살펴보아야 합니다.

    질문 내용(프롬프트)

    Q01: mysql에서 미리  'test01_db'와  'collection01'을 만들어야 하나? 

Q02: 위 sql 코드를 파이썬을 이용해 생성할 수는 없나?

    질문에 대한 Chat GPT 답변 결과

    • Chat GPT에게 6개의 질문을 해서 나온 답변들의 최종 결과는 아래와 같습니다.

    • Chat GPT를 이용시 반드시 GPT답변에 대한 사실 여부 확인이 필요합니다.

    • 파이썬 코딩에 대한 답변의 사실 여부는 직접 코딩을 실행해보면 나타납니다.

    • Chat GPT는 사실이 아닌 것을 마치 사실인 것처럼 답변하기도 합니다.

    Chat GPT 답변

    A01: 네, 정확히 맞습니다. 코드에서 MySQL 데이터베이스에 테이블에 데이터를 저장하기 위해서는 해당 

데이터베이스와 테이블이 사전에 생성되어 있어야 합니다.

따라서 데이터베이스와 테이블을 먼저 MySQL에서 생성하고, 그 후에 코드를 실행하여 데이터를 저장할 수 있습니다.

데이터베이스 생성 방법:

sql
Copy code

CREATE DATABASE IF NOT EXISTS test01_db;

테이블 생성 방법:

sql
Copy code

USE test01_db;

CREATE TABLE IF NOT EXISTS collection01 (
    column1 DATATYPE,
    column2 DATATYPE,
    ...
);

여기서 DATATYPE은 각 열의 데이터 유형을 나타냅니다. 실제로 사용할 열과 데이터 유형에 따라 적절하게 

수정해야 합니다.


A02: 네, 파이썬을 사용하여 SQL 쿼리를 실행하여 데이터베이스와 테이블을 생성할 수 있습니다. 이를 위해 

pymysql 라이브러리를 사용할 수 있습니다. 아래는 해당 작업을 수행하는 방법입니다.

python
Copy code
import pymysql

def create_database(cursor, database_name):
    try:
        cursor.execute(f"CREATE DATABASE IF NOT EXISTS {database_name};")
        print(f"Database '{database_name}' created successfully.")
    except Exception as e:
        print(f"Error creating database '{database_name}': {e}")

def create_table(cursor, database_name, table_name):
    try:
        cursor.execute(f"USE {database_name};")
        cursor.execute(f"CREATE TABLE IF NOT EXISTS {table_name} (column1 DATATYPE, column2 DATATYPE, ...);")
        print(f"Table '{table_name}' created successfully in '{database_name}' database.")
    except Exception as e:
        print(f"Error creating table '{table_name}' in database '{database_name}': {e}")

# DB 연결
con = pymysql.connect(user='root',
                      passwd='1234',
                      host='127.0.0.1',
                      charset='utf8')
cursor = con.cursor()

# 데이터베이스와 테이블 생성
database_name = 'test02_db'
table_name = 'collection01'
create_database(cursor, database_name)
create_table(cursor, database_name, table_name)

# 연결 종료
con.close()

위 코드에서 DATATYPE 부분을 적절한 데이터 유형으로 변경해주어야 합니다. 데이터베이스 및 테이블이 이미 

존재하는 경우 IF NOT EXISTS를 사용하여 무시하도록 설정되어 있습니다.

    답변 결과 디버깅 및 수정

    • 위 코드에서 DATATYPE 부분을 적절한 데이터 유형하라고 GPT가 제안하여

    column1 FLOAT, column2 VARCHAR(30)와 같이 변경했습니다.

    • 전형적인 파이썬 코딩 결과물입니다.

    • 함수설정 -> 함수에 입력값 정의 -> 함수 실행의 절차로 코딩이 구성되어 있습니다.

    • 함수 만드는 방법도 try와 excep를 사용하여 에라를 체크하게 하였고 잘 실행되면

    성공적으로 실행되었다는 메세지도 출력합니다.

    완성된 최종 프로그램

    import pymysql

# 데이터베이스 생성 함수 
def create_database(cursor, database_name):
    try:
        cursor.execute(f"CREATE DATABASE IF NOT EXISTS {database_name};")
        print(f"Database '{database_name}' created successfully.")
    except Exception as e:
        print(f"Error creating database '{database_name}': {e}")

# 테이블 생성 함수 
def create_table(cursor, database_name, table_name):
    try:
        cursor.execute(f"USE {database_name};")
        cursor.execute(f"CREATE TABLE IF NOT EXISTS {table_name} (column1 FLOAT, column2 VARCHAR(30));")
        print(f"Table '{table_name}' created successfully in '{database_name}' database.")
    except Exception as e:
        print(f"Error creating table '{table_name}' in database '{database_name}': {e}")

# DB 연결
con = pymysql.connect(user='root',
                      passwd='1234',
                      host='127.0.0.1',
                      charset='utf8')
cursor = con.cursor()

# 데이터베이스와 테이블 이름 설정 
database_name = 'test01_db'
table_name = 'collection01'

# 데이터베이스와 테이블 생성 함수 실행
create_database(cursor, database_name)
create_table(cursor, database_name, table_name)

# 연결 종료
con.close()

    결어

    • MySql에서 DB와 테이블을 생성 자동화프로그램을 만들어 보았습니다.

    • Chat GPT는 2번의 질문으로 코딩 결과를 산출했습니다.

    • 코딩의 결과물이 매우 전형적인 형식을 이루고 있습니다.

    • 이제 문제없이 실행되면 exe 프로그램으로 전환하여 자동화 프로그램을 실행해 봅시다.
