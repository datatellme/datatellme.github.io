    텍스트 파일 입력기(gppj01_0402)

    들어가며(Introduction)

    목표(Goal)

    • 텍스트 파일 입력기는 책이나 컨텐츠를 텍스트 파일로 생성하기 위해서 만든 것입니다.
    • 웹 서핑이나 Chat GPT와 대화 하면서 습득한 자료들을 쉽게 텍스트 파일로 저장할 수 있게 하기위해서 만들었습니다.

    목적(Why)

    • 텍스트 파일 입력기는 여러가지 형태의 문서 입력기의 기본이 되는 프로그램 입니다.
    • 앞으로 다루게 될 Rmd 파일 입력기나 md 파일 입력기 등 파일 형태를 달리하더라도 응용해서 사용이 가능합니다.

    방법(How)

    질문(프롬프트)

    • 먼저 Chat GPT에게 '너는10년차 파이썬 프로그래머처럼 PEP 8스타일 가이드를 따라'라고 페르소나를 선언해 줍니다.
    • 다음으로 어떠한 작업 수행인지 알려 줍니다. 아래와 같이 'tkinter로 GUI 창을 만들어서 텍스트 파일을 작성하여 텍스트 파일로 저장하는 기능을 수행하는 코드'를 작성해 달라고 합는 것입니다.
    • 마지막으로 참고할 예시 파이썬 코딩을 알려 줍니다. 
    Q01: 너는 10년차 파이썬 프로그래머처럼 PEP 8스타일 가이드를 따라 아래 파이썬 코드를 참조해서 tkinter로 GUI 창을 

만들어서 텍스트 파일을 작성하여 텍스트 파일로 저장하는 기능을 수행하는 코드를 수정해줘.  

예를 들어 저장할 내용의 텍스트 파일은 directory_out = r'C:\gppj\gppj01\i'에 book_output.txt로 만들어 주는거야.
산출된 텍스트 파일을 저장할 수 있는 버튼을 생성해줘. 
저장이 완료되면 tkinter 메세지 박스로 '저장이 완료 되었습니다.'라고 표시해줘.

다음

import os
import sys
import time
from datetime import datetime
import codecs


# inf_output에 프린트할 내용
text_to_print = ''' tkinter로 GUI 창으로 입력할 내용 작성 '''


# inf_output 입력 파일과 출력 디렉토리 경로 설정
# output_directory01의  C:\gppj\gppj01\i는 작업 폴더로 수정해야 함.
output_directory01 = r'C:\gppj\gppj01\i'
output_file01 = 'inf_output.txt'
output_path01 = os.path.join(output_directory01, output_file01)

# 시작 시간 세팅 
start_time = time.time()
original_stdout = sys.stdout

# 주제 텍스트 파일에 작성
with open(output_path01, 'a', encoding='utf-8') as file:
    sys.stdout = file
    print(text_to_print)
    sys.stdout = original_stdout


# 작업에 걸린 시간 텍스트 파일에 작성
end_time = time.time()
elapsed_time = end_time - start_time
with open(output_path01, 'a', encoding='utf-8') as file:
    file.write(f"걸린 시간: {elapsed_time:.2f} seconds\n")

# 현재 날짜와 시간을 가져옴
current_datetime = datetime.now()

# 날짜와 시간을 각각 변수에 저장
year = current_datetime.year
month = current_datetime.month
day = current_datetime.day
hour = current_datetime.hour
minute = current_datetime.minute
second = current_datetime.second

# 현재 작업한 시간 텍스트 파일에 작성
with open(output_path01, 'a', encoding='utf-8') as file:
    file.write(f"작성 년도: {year}.")
    file.write(f"{month}.")
    file.write(f"{day}.\n")
    file.write(f"현재 시간: {hour}:{minute}:{second}\n\n")

    결과(What)

    • 위에 지시한 프롬프트 질문을 카피해서 Chat GPT에게 물어보면 물어 볼때마다 조금씩 다른 내용의 답변을 하는 것을 볼 수 있습니다.
    • 그래서 우리는 최소한의 파이썬 코딩을 해석할 정도의 지식이 필요합니다.
    • 파이썬 문법을 알고 있어야 결과를 해석하고 오류를 수정할 수 있는 것입니다.

    챗 GPT 답변

    • 아래는 코드에 변경 사항을 적용한 완성된 버전입니다.

    import tkinter as tk
from tkinter import messagebox
import os
from datetime import datetime


def save_text_to_file():
    text_content = text_entry.get("1.0", tk.END)  # 텍스트 입력란의 내용 가져오기

    # 출력 디렉토리 및 파일 경로 설정
    output_directory = r'C:\gppj\gppj01\i'
    output_file = 'book_output.txt'
    output_path = os.path.join(output_directory, output_file)

    # 텍스트 내용을 파일에 저장
    with open(output_path, 'w', encoding='utf-8') as file:
        file.write(text_content)

    # 현재 시간 가져오기
    current_time = datetime.now().strftime("%Y-%m-%d %H:%M:%S")

    # 작성 시간을 파일에 추가
    with open(output_path, 'a', encoding='utf-8') as file:
        file.write(f"작성 시간: {current_time}\n")

    # 저장 완료 메시지 출력
    messagebox.showinfo("저장 완료", "저장이 완료되었습니다.")


# Tkinter GUI 생성
root = tk.Tk()
root.title("텍스트 파일 저장기")
root.geometry("400x300")

# 텍스트 입력 위젯 생성
text_entry = tk.Text(root)
text_entry.pack(fill="both", expand=True)

# 저장 버튼 생성
save_button = tk.Button(root, text="저장", command=save_text_to_file)
save_button.pack()

root.mainloop()
    - 이제 저장된 텍스트 파일에 작성 시간이 기록됩니다.

    결어

    • 간단한 텍스트 입력기를 만들어 보았습니다.
    • 프롬프트 질문의 절차를 페르소나 선언, 작업수행 내용 기술, 참고 파이썬 코딩 제시 등의 순서로 구성하여 작성하였습니다.
    • 질문 결과는 각자가 조금씩 차이가 날 수 있으며 위의 결과와 비고해보시기 바랍니다.
    • 반드시 결과는 파이썬을 실행해서 잘 실행되는 확인하여야 합니다.
    • 실행하는 방법은 RStudio의 'ppj01.Rproj'를 실행하여 RStudio를 열고 Console 옆의 Terminal을 클릭하여 PowerSell로 들어갑니다.
    • PowerSell에서 'conda activate myfirstenv' , 'cd p', 'python gppj01_0402.py' 엔터 치면 실행됩니다.
