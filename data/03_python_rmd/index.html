    R Markdown에서 Python을 사용

    • R Markdown에서 Python을 사용하려면 코드 청크안에서 {reticulate}이라는 R 패키지 로드해야 합니다.
    • R에서 그래프를 그리려면 {ggplot2} 패키지를 사용할 수 있습니다.
    • Python의 pandas 데이터를 R의 ggplot2 에서 연결하여 그래프로 나타내는 것이 가능합니다.

    .Rmd 파일에서 Python 사용

    패키지 로드

    R과 마찬가지로 .Rmd 파일에서 Python을 사용할 수 있습니다. 여기에서 Python을 가져옵니다. 데이터를 Python으로 가져오는 데 필요한 패키지인 pandas. pandas는 데이터 읽기, 랭글링 및 간단한 데이터 시각화를 할 수있는 Python 패키지입니다. R 패기지 {tidyverse}와 비슷한 위치를 가지는 meta-package 역할을 수행합니다(그러나 pandas는 메타 패키지는 아닙니다) pandas에 대해 자세히 알아보려면 다음을 참조하세요. “10 minutes to pandas” 문서에서 확인하십시오. https://prog-learn.mds.ubc.ca/en.

    import pandas as pd

    데이타 로드

    타이타닉 데이터(이 데이터의 data 디렉토리에 있음)를 로드합시다.

    titanic  = pd.read_csv("data/titanic.csv")

    그래프 data

    • Python and pandas를 이용

    먼저 빠르고 간단한 pandas 그래프 기능을 사용해 보겠습니다.

    titanic.plot.scatter(x='age', y='fare', alpha=0.3)

    Python에서 pandas를 사용하여 그래프를 그리기 시작하는 방법에 대해 자세히 알고 싶습니다. 시작하려면 이 페이지를 참조하십시오. https://pandas.pydata.org/pandas-docs/stable/user_guide/visualization.html

    • R과 ggplot2 이용

    R에서 이 R Markdown 문서의 Python 환경에 액세스할 수도 있습니다. 이를 통해 Python 객체에 R의 기능을 적용할 수 있습니다. 우리는 이를 수행하고 {ggplot2}를 사용하여 데이터 시각화를 생성하는 py$obbject 구문이 사용됩니다.

    ggplot2::ggplot(py$titanic, aes(x = age, y = fare)) +
  geom_point()
