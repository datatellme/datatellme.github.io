  • lang=’eng+kor’은 영어 + 한글 인식을 위해 설정한 것입니다. → tesseract에 kor 언어 데이터가 설치되어 있어야 작동합니다.

  • 이미지 이름이 test001.jpg면 → test001.txt로 저장됩니다.

  • 100개마다 진행 상황을 출력합니다.

  • 폴더 안의 모든 JPG 이미지를 각각 하나의 TXT 파일로 저장하는 파이썬 코드의 완성본입니다.

from PIL import Image
import pytesseract
import os
from tqdm import tqdm

# 원본 이미지 폴더
path = r"D:\vFlat\man_A02"

# TXT 저장 폴더
ConvertedToTxtPath01 = r"D:\vFlat\Atxt02"

# TXT 저장 폴더가 없다면 생성
os.makedirs(ConvertedToTxtPath01, exist_ok=True)

# 파일 목록 불러오기 및 정렬
file_list = sorted(os.listdir(path))

# 처리 시작
for idx, filename in tqdm(enumerate(file_list, start=1)):
    if filename.lower().endswith(('.jpg', '.jpeg', '.png')):
        if idx % 100 == 0:
            print(f"Progress: {idx}/{len(file_list)}")

        img_path = os.path.join(path, filename)
        img = Image.open(img_path)
        img_rgb = img.convert('RGB')

        # OCR 수행 
        pytesseract.pytesseract.tesseract_cmd = r"C:\Program Files\Tesseract-OCR\tesseract.exe"
        text = pytesseract.image_to_string(img_rgb, lang='eng+kor')

        # 결과 저장할 TXT 파일 경로 설정
        txt_filename = os.path.splitext(filename)[0] + ".txt"
        txt_path = os.path.join(ConvertedToTxtPath01, txt_filename)

        # 텍스트 파일로 저장
        with open(txt_path, 'w', encoding='utf-8') as f:
            f.write(text)

print("모든 이미지가 텍스트 파일로 저장되었습니다.")

4 단계 작업

  • D:_Atxt02 폴더 안의 모든 *.txt 파일을 하나로 합치는 batch 스크립트

  • 파일명 순서대로 (오름차순) 정확하게 합치고 싶다면, copy *.txt는 안 되고, for문을 써야 해요.

  • (echo?) off : 명령어 실행 내용을 화면에 표시하지 않음 (깔끔하게)

  • cd /d D:_Atxt02 : 해당 폴더로 이동 (/d는 드라이브까지 이동할 때 필요)

  • copy /b *.txt All.txt : 모든 .txt 파일을 이진 모드(binary mode)로 All.txt 파일로 합칩니다. (/b 옵션을 넣으면 깨짐을 방지할 수 있어요.)

  • dir /b /on *.txt : 파일 이름 기준 오름차순 정렬 목록을 가져옴

  • for /f : 정렬된 파일 목록을 하나씩 순서대로 처리

  • type “파일명” >> “All.txt” : 하나씩 내용을 이어붙임 (append)

  • pause : 명령어 실행 후 결과를 볼 수 있도록 멈춤

@echo off
cd /d D:\vFlat\Atxt02

:: 폴더가 없다면 먼저 생성
if not exist "D:\vFlat\Atxt03" (
    mkdir "D:\vFlat\Atxt03"
)

:: All.txt 초기화 (기존 파일 있으면 삭제)
if exist "D:\vFlat\Atxt03\All.txt" del "D:\vFlat\Atxt03\All.txt"

:: 파일명을 오름차순으로 하나씩 읽어서 All.txt에 추가
for /f "delims=" %%i in ('dir /b /on *.txt') do (
    type "%%i" >> "D:\vFlat\Atxt03\All.txt"
)

pause
