  • VideoFileClip으로 mp4들을 모두 읽어온 뒤

  • concatenate_videoclips로 한 번에 이어붙여서

  • 하나의 최종 .mp4 파일로 저장해야 합니다.

  • concatenate_videoclips 사용 여러 개 mp4 파일을 하나로 이어붙임

  • for 반복하면서 파일마다 저장 ❌ 전부 메모리에 읽은 다음 한번에 합쳐서 저장

  • method=“compose” 사용 해상도/코덱이 서로 달라도 합칠 수 있게 함

  • 모든 파일 clip.close() 메모리 릴리즈 (안 하면 누수 위험 있음)

from moviepy.editor import VideoFileClip, concatenate_videoclips
import os
from tqdm import tqdm

# 원본 Mp4 파일이 있는 폴더
path = r"D:\data\moviepy\20230501"

# 합쳐진 Mp4 파일을 저장할 폴더
output_path = r"D:\data\moviepy\20230501_all"

# Mp4 저장 폴더가 없다면 생성
os.makedirs(output_path, exist_ok=True)

# Mp4 파일 목록 불러오기 및 정렬
file_list = sorted([f for f in os.listdir(path) if f.lower().endswith('.mp4')])

# 모든 클립 읽기
clips = []

print("📥 Mp4 파일 읽는 중...")
for file_name in tqdm(file_list, desc="MP4 읽는 중"):
    mp4_file = os.path.join(path, file_name)
    try:
        clip = VideoFileClip(mp4_file)
        clips.append(clip)
    except Exception as e:
        print(f"오류 발생: {file_name} - {e}")

# 클립 이어붙이기
if clips:
    print("🛠️ 클립 합치는 중...")
    final_clip = concatenate_videoclips(clips, method="compose")  # method="compose"로 서로 다른 해상도도 가능

    # 최종 파일 저장
    output_file = os.path.join(output_path, "all.mp4")
    final_clip.write_videofile(output_file, codec="libx264", audio_codec="aac", preset="ultrafast")

    # 모든 클립 닫기
    for clip in clips:
        clip.close()

    print("✅ 모든 Mp4 파일이 all.mp4로 합쳐졌습니다.")
else:
    print("❌ 합칠 mp4 파일이 없습니다.")

  • 파일 크기가 커질 수 있어요. (compose 쓸 때)

  • 만약 모든 소스 mp4가 해상도, 코덱이 같으면 method=“chain”으로 더 빠르게 합칠 수 있습니다:

final_clip = concatenate_videoclips(clips, method="chain")
  • preset=“ultrafast”를 주면 저장 속도가 빨라지지만, 파일 용량은 살짝 늘어납니다.
